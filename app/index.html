Wire Themes

Intro

This library is dealing only with the colors. Different kind of colors but only colors.

You should include only the wire-theme.less into your project. The rest of the styles that you can find here are only serve as an example.

.text-theme

This is just a simple text that is not affected by the theme colors.

This text is affected by the theme color because it has the text-theme class.

It could also be just part of the text.

The links are automatically taking the theme color without extra classes. But they are a bit darker than the normal text.

They are darker because if the whole text is themed, then you should be able to find the link. If it's not underlined.

.bg-theme

Backrounds with colors! Yeah..

If you need something that has a background color of the theme. Then just use the bg-theme class on that element.

This thing has some special powers though. When you are using the text-theme here, it will be white.

The links will be also highlighted automatically.

.btn-theme & .btn-outline-theme

If you need a button that matches the theme color, then add the btn-theme class. Please note that only the colors are being manipulated by this module.


When we have buttons on something that has already a background, then the buttons are a bit darker.

.btn

For better handling of buttons add the btn class.

.border-theme

If something has border you can add a border-theme to change the border color.

When the same border is on a background then it will be slighlty darker. But maybe it should be white?!?

.range-theme

SVG (inline)

For inline SVG simply add the class text-theme and it will be treated as a text.

And the same SVGs on a background of a theme.
